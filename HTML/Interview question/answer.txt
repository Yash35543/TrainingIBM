1. How will you start a ordered list from 10 ?

3. Name any 5 Html5 tags
<header>, <footer>, `<nav>`, <article>, and <video>.

4. What is dns and when it come in picture
Domain Name System. It acts as the internet's phonebook, translating human-friendly domain names (like google.com) into IP addresses the moment you type a URL into a browser.

5. What is Markup
A system of formatting and structuring text using tags (like <h1> or <p>) to instruct a browser on how to display content.

6. SEO {Knowledge of basics}
Search Engine Optimization. The practice of structurally optimizing web pages (using semantic tags, alt text, proper headers, and meta tags) to rank higher in search engine results.

7. Inspect Elements
A developer tool built into web browsers (usually opened with F12) used to view, test, and debug HTML, CSS, and JavaScript in real-time.

8. Explain what is webpage designing
The process of planning, creating, and establishing the visual appearance, user interface (UI), layout, and overall aesthetics of a page on the internet.

9. How will u open another windows using anchor tag
By adding the target="_blank" attribute: <a href="URL" target="_blank">Text</a>.

10. what does viewport meta tag do
It controls the page's dimensions and scaling on mobile device screens, making the webpage responsive (content="width=device-width, initial-scale=1.0").

11. What are Attributes in html
Special modifiers that provide extra information or properties about an HTML element, placed inside the opening tag (e.g., href in <a>).

12. What is dns
The Domain Name System, which translates human-readable domain names into computer-readable IP addresses.

13. what is ip address
Internet Protocol Address. A unique numerical label assigned to every device connected to a computer network to identify its location.

14. what is localhost
The default hostname pointing back to your local computer (loopback address, traditionally 127.0.0.1).

15. HTML5 vs HTML4
HTML5 introduced native audio/video playback, modern structural tags (<nav>, <header>), canvas graphics, local storage, and forms without relying on third-party plug-ins like Flash.

16. what are semantic tags
Tags that clearly communicate the purpose and meaning of their content to both the developer and the browser (e.g., <article>, <main>).

17. what are the new features in HTML5?
Semantic tags, native video/audio tags, <canvas>, SVG support, geolocation APIs, web storage (localStorage), and new form inputs (email, date).

18. Difference between div and span
<div> is a block-level element (starts on a new line, spans full width). <span> is an inline element (does not break lines, wraps only its immediate content).

19. Difference between class and IDs
An ID must be completely unique to a single element per page (id="main-nav"). A Class can be reused across multiple elements on the same page (class="btn").

20. What are self closing tags
Tags that do not enclose content and therefore do not require a separate closing tag. Examples include <img>, <br>, and <input>.

21. How many types of headings are der ?How to decide which to use where
There are 6 types (<h1> through <h6>). Decide their usage strictly hierarchically based on the structure and priority of the topic matter, not for text sizing.

22. Difference between block and inline
Block elements take up the full available horizontal width and start on a new line (e.g., <p>). Inline elements sit side-by-side with other content and only take up the width of their actual content (e.g., <strong>).

23. Difference between em and strong
<em> applies verbal emphasis (rendered visually in italics). <strong> signals strong importance or urgency (rendered visually in bold).

24. What are attributes?
Key/value properties specified within an opening tag to configure or provide metadata about an HTML element.

25. Difference between HTML tags and elements
A Tag is just the syntax opener/closer (<p> or </p>). An Element is the complete bundle consisting of the opening tag, the inner content, and the closing tag.

26. Explain br tags
The <br> tag forces a line break within text without starting a brand-new paragraph structure.

27. What are semantic tags?
Tags designed to inherently describe their internal content meaning (e.g., <aside>, <footer>) rather than acting as generic style-less placeholders.

92. What is the relationship between the border and rule attributes?

93. What is Cell Spacing and Cell Padding?
Cell Padding defines the empty buffer space between text content and its surrounding border inside a cell. Cell Spacing manages the gap separating individual cell borders from each other.

94. What is difference between HTML and XHTML?
HTML allows flexible parsing rules (tags can skip closing). XHTML stems directly from XML specifications, mandating lowercase element strings, strict tag termination, and perfectly nested syntax.

95. How many types of CSS can be included in HTML?

96. What are logical and physical tags in HTML?
Physical tags explicitly instruct a browser on visual styling rules (e.g., <i> for italic). Logical tags define context importance (e.g., <strong> for emphasized), delegating final presentation styles to the browser or CSS.
